Advanced Down Payment from sale order

This module provides easy feature to register down payment against the sale order with respective taxes. User will be able to create down payment lines for each sale order line. If down payment is registed with percentage amount then same percentage will be given to subtotal of sale order line and if down payment is registed with fixed amount then fixed amount will be divied to each line based on weightage of subtotal.

Features

  • Register down payment with respective taxes and flexible amount.
  • Register separate down payment for each sale order line.
  • Equally distributed percentage based down payment.
  • Weightage distributed fix amount based down payment.

Highlights

  • Register down payment from invoice

  • Breakdown of percentage based down payment based on each sale order line.

  • Draft Invoice with different applied taxes from sale order line.

  • Down Payment lines added to sale order.

  • Breakdown of fix amount based down payment based on each sale order line and total amount of sale order.

  • Draft Invoice with fixed amount of down payment and different applied taxes from sale order line.
