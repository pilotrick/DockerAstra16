Download tourtravel_management_aagam module
1. Go to tourtravel_management_aagam/jobcard folder
2. Cut JobCard folder and move these folder into your addons path / custom addon path
3. Replace 'depends': ['base'] line with  'depends': ['base','product', 'account', 'jobcard',],
4. Go to apps and update applist.
5. Click on install tourtravel_management_aagam. 
6. Also after performing this restart the odoo services.
