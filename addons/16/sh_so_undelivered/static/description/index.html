Sales Order Undelivered

If you want to get an undelivered sales order? Here we build a module that can help to find all undelivered sales orders easily, very easy to find undelivered sales order from one click with list view and form of the sales order. You can easily group by undelivered order by name, customer, order no, etc.

Features

Easy to Track all Undelivered Sales Orders in one click.

Easy to display all Undelivered sales order records with form and list view.

Easy to filter all undelivered records with order date.

Easy to Group by all undelivered records with a partner, product, order date etc.

Easy to install and use it.

No other configuration required.

This app is compatible with Odoo's Multi-Company Feature.
