Purchase Delivery Method

Description

This module add delivery method in the purchase order and fill the value from the vendor selected.

Functionality

This module is used to add delivery method in the purchase order and set default value from the vendor selected and also if purchase order created from sales order.

1.Selected Purchase Delivery Method for the vendor.


2.In the selected product, the vendor will be selected based on the lowest price.


3.In a purchase order, the delivery method will be propagated based on the vendor selected.


4.Now, on hand is 0 for the product, so it will create the purchase order on SO confirm.


5.The purchase order propagated the delivery method from the vendor.
