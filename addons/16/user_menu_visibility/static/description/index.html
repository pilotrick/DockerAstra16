Hides menu entries and entire applications from App Menu

Allows admins to restrict user access to menus or apps menu entries.




  • The User Menu Visibility module, accessible through the "Settings" â†’ "Users" section, introduces a new feature called "Hidden Menus."



  • By clicking on "Add a line", a pop-up will show you a list of menus and a search bar. Administrators can easily select specific menus or entire apps to be hidden from regular users.

  • Apps found in the "Hidden Menus" pop-up window do not have parent menus. That is how you can differentiate the "apps" from the "menu items"



  • On the above example, upon logging out and logging in, the target user will no longer be able to see the "Contacts" App, but will still be able to access contact records as long as they are linked in a different menu.
