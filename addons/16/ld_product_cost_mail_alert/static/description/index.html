Cost Email Alert

This Module have the functionality by which user can control their quotation cost prices.
During quotation creation, If system found any unit price is less then its configured cost price, system will notify the user by email with order number and the item name.

Inside General Settings -> Enable Send Cost Alert and select the user [You can choose multiple users as well]




Configured the products - [Add cost price of the products with some on-hand quantities]





Create quotation, add those products and add unit prices as well, confirm that quotation - [System will trigger email to the configured users once found any unit price less then its configured cost price]



Check Email - [Configured users will receive email with quotation reference number and the item for which the price is less then cost price]
